# Terrafrom

This directory contains the Terraform configuration for provisioning the AWS infrastructure required for the VProfile application.

## Modules

The infrastructure is modularized into the following components:

- `modules/`: Contains the reusable Terraform modules for various AWS resources (VPC, EC2, RDS, etc.).

## Usage

1. **Initialize Terraform**:
   ```bash
   terraform init
   ```

2. **Review the Plan**:
   ```bash
   terraform plan
   ```

3. **Apply the Configuration**:
   ```bash
   terraform apply
   ```

## Inputs

Refer to `variables.tf` for the list of input variables that can be customized.

## Outputs

Refer to `outputs.tf` for the outputs generated by the Terraform run.
